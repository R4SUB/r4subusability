# Print a Usability Result Summary

Print a Usability Result Summary

## Usage

``` r
# S3 method for class 'usability_result'
print(x, ...)
```

## Arguments

- x:

  A `usability_result` object returned by
  [`usability_summary()`](https://r4sub.github.io/r4subusability/reference/usability_summary.md).

- ...:

  Ignored.

## Value

Invisibly returns `x`. Called for its side effect of printing a summary
of usability indicator scores (pass/fail/warn counts and overall
proportion) to the console.
