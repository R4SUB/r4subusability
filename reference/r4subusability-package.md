# r4subusability: Usability Indicators for Clinical Submission Readiness

Quantifies reviewer-experience and usability indicators for clinical
regulatory submissions. Assesses variable label quality, Define-XML
(Extensible Markup Language) completeness, dataset annotation coverage,
and reviewer guide presence. Emits standardised R4SUB (R for Regulatory
Submission) evidence table rows via 'r4subcore' for integration with
'r4subscore' and 'r4subprofile'.
